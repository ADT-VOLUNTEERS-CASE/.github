# Решение кейса центра городских волонтеров учениками ГБНОУ "Академия технического творчества и цифровых технологий"
С заданием кейса можно ознакомится [тут](./../documentation/case-requirements.md)
## Путевая карта проекта
- [ ] Проведение исследования существующих систем учёта волонтёрской деятельности
- [ ] Анализ потребностей целевой аудитории (волонтёры, координаторы, администраторы/сотрудники) и определение требований к функционалу системы
- [ ] Проектирование архитектуры системы
- [ ] Разработка базы данных для хранения информации о волонтёрах и мероприятиях
- [ ] Реализация системы учёта волонтёрских часов и активностей
- [ ] Интеграция с Телеграм API для работы бота
- [ ] Разработка аналитического модуля с системой отчётности
- [ ] Обеспечение безопасности данных и защиты от несанкционированного доступа.
- [ ] Создание возможности выгрузки публичного рейтинга волонтеров (количества часов).
