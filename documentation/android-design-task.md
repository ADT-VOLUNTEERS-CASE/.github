# Раздел "Поиск"
По умолчанию приложение открывается на этом экран
## Список/Карта мероприятий
- Переключение отображения список/карта
	- В списке отображаются карточки мероприятий 
		- Обложка
		- Название
	    - Дата и время
        - Адрес
	    - Кол-во мест
	- На карте отображаются пины с мероприятиями
- Фильтры мероприятий (всплывающее окно фильтров)
	- Дата
	- Район
	- Тематика
- Строка поиска для поиска текстом (к нему соответственно окно поиска или отдельный экран)
## Экран мероприятия
- Название, обложка
- Дата, время, адрес.
- Описание.
- Контакты координатора.
- Количество и доступность мест.
- Кнопка «Записаться» / «Отменить участие».
# Раздел "Активность"
## Экран мои мероприятия
- Небольшой блок статистики с переходом на экран статистики
- Карточки предстоящих мероприятий
- Карточки прошедших мероприятий
## Экран статистики
- Часы по типам мероприятий
- Активность за последний месяц
- Возможно что-то ещё
# Раздел "Обо мне"
## Экран профиль
- Личная информация
	- ФИО
	- Контакты
	- Интересы
	- Кнопка «Редактировать профиль»
- Кнопка настройки
## Экран редактирование профиля
- Поля редактирования
	- ФИО
	- Телефон
	- Почта
	- Интересы (тегами)
- Кнопки Сохранить и Отмена
## Экран Настройки
- Уведомления в телеграмме (вкл/выкл)
- Выход из аккаунта
